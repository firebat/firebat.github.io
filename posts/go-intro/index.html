Go简明教程

2022/08/22

本教程为《Go语言圣经》内容简化，作为开发同学快速上手用。

语言基础

  • 没有异常、宏
  • 常量编译时计算
  • 包、变量 声明后必须使用，否则报错
  • 使用_丢弃数据，防止编译报错（变量、包）
  • 包初始化默认执行init()
  • 首字母大写可外部访问，否则仅内部访问
  • switch默认break，相邻case执行需加fallthrough (用得少）

Defer

  • 延时处理函数，函数执行后、内存释放前执行
  • 可以写多个，先入后出
  • os.Exit、log.Fatal 会导致Defer函数不执行，慎用

func ReadFile(filename string) {
  f, err := os.Open(filename)
  if err != nil {
    return nil, err
  }
  defer f.Close()
  return ReadAll(f)
}
